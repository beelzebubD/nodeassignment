Instructions

1.To create a user just type the username and pasword in the given field and push create user button.Same username can be created as it is not constrained.

2.To delete a user just type the username and pasword in the given field and push delete user button.All the username having the specified username and password will be deleted. You should know both the username and password to delete it from the database.

3.To update a user's password just type the username and the new password pasword in the given field and push update user password button. You shoud know the username whose password is to be updated.

4.To get a user information just type the username in the given field and push get user button.
